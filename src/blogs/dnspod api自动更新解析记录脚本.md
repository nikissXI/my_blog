<p>如果家里网络有公网IP，但是IP每天都不一样，那么D整个DNS服务就很有必要了<br/>下面操作是在Ubuntu下进行</p><p>先看一看dnspod api的官方文档 <b><u><a href="https://support.dnspod.cn/api/" target="_blank">点击跳转</a><br/></u></b></p><p>第一步先安装jq，用于解析json数据</p><pre><code class="Bash"><xmp>sudo apt install jq</xmp></code></pre><p>第二步去获取API Token<b><u><a href="https://www.dnspod.cn/console/user/security" target="_blank"><br/>点击跳转</a></u></b><br/>创建后就获得API的ID和Token，因为只显示一次，所以Token的值自己记着</p><p>第三步获取域名的ID<br/>假设API的ID是123456，Token是aabbcc112233，login_token就是123456,aabbcc112233<br/>然后执行以下命令获取域名ID</p><pre><code class="Bash"><xmp>curl 'https://dnsapi.cn/Domain.List' -d 'login_token=123456,aabbcc112233&format=json'rn</xmp></code></pre><p>第四步获取子域名ID<br/>假设第三步获取的域名ID是778899，然后执行以下命令获取子域名ID</p><pre><code class="Bash"><xmp>curl 'https://dnsapi.cn/Record.List' -d 'login_token=123456,aabbcc112233&format=json&domain_id=778899'</xmp></code></pre><p>假设第四步获取到的子域名ID为<br/>a记录的ID是12<br/>b记录的ID是34<br/>c记录的ID是56<br/>然后要对这三个子域名解析记录更新IP<br/>脚本如下</p><pre><code class="Bash"><xmp>#!/bin/bashrnrntime=$(date "+%Y-%m-%d %H:%M:%S")rnoldIPFile=./oldip.txtrnlogFile=./updatelog.txtrnLOGIN_TOKEN=123456,aabbcc112233rnRECORD_ID=12,34,56rnrnfunction updateIp() {rn  result=$(curl -X POST https://dnsapi.cn/Batch.Record.Modify -d "login_token=${LOGIN_TOKEN}&format=json&record_id=${RECORD_ID}&change=record_type&change_to=A&value=${myip}" | jq .status.code)rn  code="\"1\""rn  if [ "$result" = "$code" ];rn   thenrn       echo "${time} 更新成功 ${myip}" >> $logFilern       echo "$myip" > $oldIPFile rn   elsern       echo "更新失败,代码: $result" >> $logFilern       exit 0rn  firn}rnrnmyip=$(curl https://ipinfo.io/ 2>> /dev/null | grep '\"ip\"' | awk -F '"' '{print $4}')rnecho "当前ip:$myip"rnrnif [ ! -f "$oldIPFile" ]; thenrn  updateIprnelsern  oldip=$(cat $oldIPFile)rn  if [ "$myip" = "$oldip" ]; thenrn    exit 0rn  elsern    updateIprn  firnfi</xmp></code></pre><p>假设要每分钟执行一次<br/>添加定时任务</p><pre><code class="Bash"><xmp>crontab -e</xmp></code></pre><p>然后在最后一行增加(假设我脚本放在/home目录下，名字叫dns.sh)，编辑后保存</p><pre><code class="Bash"><xmp>* * * * * cd /home && bash dns.sh</xmp></code></pre><p>最后，记得给脚本x权限</p>
