<p># 使用 parted 调整分区大小<br/>parted /dev/vda<br/>(parted) rm 2 &nbsp; &nbsp;（删除后如果有提示使用中，Ignore就行）<br/>(parted) mkpart primary 1G 100% &nbsp; &nbsp; （1G是前面分区的空间大小）<br/>(parted) quit<br/><br/># 调整物理卷<br/>pvresize /dev/vda2<br/><br/># 扩展逻辑卷<br/>lvextend -l +100%FREE /dev/mapper/ao-root<br/>lvextend -l +100%FREE /dev/mapper/ao-home<br/><br/># 扩展文件系统<br/># ext4 文件系统<br/>resize2fs /dev/mapper/ao-root    # 根路径用这个<br/>resize2fs /dev/mapper/ao-home    # 非根路径用这个<br/><br/># xfs 文件系统   公司业务系统是这个 <br/>xfs_growfs /&nbsp; &nbsp;# 根路径用这个<br/>xfs_growfs /dev/mapper/ao-home    # 非根路径用这个<br/><br/># 验证扩展结果<br/>lsblk<br/></p>
