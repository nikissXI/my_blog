**按步骤进行，不能同时进行**

# 配置自定义时间服务器(如果时间能正常同步就不用设置这个)
```plaintext
ntp.aliyun.com
time.windows.com
```

# 更新系统软件
更新完成后，重启电脑
```bash
sudo apt update
sudo apt upgrade
sudo dpkg --configure -a
```

# 安装微信
安装完成后，在菜单里找，然后添加桌面快捷方式
```bash
sudo apt install wechat
```

# WPS？
如果需要用文档编辑器，卸载系统原来的wps，安装厂家给的政企版wps，自带的得激活（给钱！）
