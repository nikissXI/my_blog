<p>配置网络，联网<br/>下面每一步按步骤进行，不能同时进行<br/><br/>1. 配置自定义时间服务器 time.windows.com<br/>(如果时间能正常同步就不用设置这个)<br/><br/>2. 更新系统软件<br/>sudo apt update<br/>sudo apt upgrade<br/>sudo dpkg --configure -a<br/>更新完成后，重启电脑，然后<br/></p><p><br/>3. 安装微信，使用命令sudo apt install wechat<br/>安装完成后，在菜单里找，然后添加桌面快捷方式</p><p><br/></p><p>4. 如果需要用文档编辑器，卸载系统原来的wps，安装厂家给的政企版wps，自带的得激活（给钱！）</p>
